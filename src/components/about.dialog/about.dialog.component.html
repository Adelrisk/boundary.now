About the Project

Getting the geographic boundary of somewhere is sometimes painful because

  • Many geocoding APIs don't provide boundary output
  • Some geocoding APIs limit the use of its boundary output
  • Official data sometimes doesn't exist
  • Open dataset is usually packaged with more data than actually needed

So the Boundary.Now project is developed to extract the place boundary from the results of Nominatim API, a geocoder based on the world's largest geospatial database, OpenStreetMap.

Developed by Haoliang Yu and opened at GitHub
